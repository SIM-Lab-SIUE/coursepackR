# Technical Guide: RStudio, Quarto, & Survey Design

This guide covers technical setup for your APA-formatted paper and survey design best practices.

## RStudio & Quarto Setup
- Create a new RStudio Project by cloning your GitHub repository
- Install necessary R packages:
  ```r
  install.packages(c("tidyverse", "rmarkdown", "quarto", "apaTables", "readr"))
  ```
- Download APA 7th Edition CSL file from Zotero Style Repository
- Save as `apa.csl` in your project folder
- Create your `research_paper.qmd` file using the template

## Survey Design
- Use clear, unbiased questions
- Pilot test your survey
- Get instructor approval before launching

For troubleshooting, see Quarto docs and ask your instructor.
