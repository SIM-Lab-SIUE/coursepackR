Build Your GitHub Landing Page

Your GitHub profile can show a custom landing page‚Äîa README.md that lives in a special repository named exactly like your username. No code needed; it‚Äôs all point-and-click in the browser.

What You‚Äôll Make

  • A public repo named your GitHub username
  • A README.md inside it

Step 1 ‚Äî Confirm Your Username

  1. Sign in at GitHub.com.
  2. Click your avatar (top-right) ‚Üí Your profile.
  3. Note your username (the text after github.com/ in the URL).

Example: github.com/alexleith ‚Üí username is alexleith.

Use the username exactly as shown. Repo names are case-insensitive, but matching avoids confusion.

Step 2 ‚Äî Create the Special Repository

  1. Click the + (top-right) ‚Üí New repository.
  2. Repository name: type your exact username (e.g., alexleith).
  3. Set visibility to Public (required).
  4. Check Initialize this repository with a README.
  5. Click Create repository.

You now have a repo at:


github.com/<username>/<username>

This is your profile repo.

Step 3 ‚Äî Edit Your README

  1. In the new repo, click README.md.
  2. Click the pencil icon (Edit).
  3. Replace the placeholder text with something short and welcoming (see template below).
  4. Scroll down, add a brief commit message (e.g., ‚ÄúCreate profile README‚Äù), and click Commit changes.

Visit your profile: click your avatar ‚Üí Your profile. You should see your README content at the top.

Troubleshooting

  • README not showing?

    • Repo must be named exactly your username.
    • Repo must be Public.
    • File name must be README.md at the root of the repo (not in a folder).

  • Can‚Äôt find ‚ÄúNew repository‚Äù? Click the + in the top-right ‚Üí New repository.
