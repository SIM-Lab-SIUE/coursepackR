# Final Project Template Guide

This folder contains all the files you need to start your Graduate Research Paper for MC501.

## How to Use This Template

1. **Create a new RStudio Project** in a fresh folder for your final paper.
2. **Run the package function** to copy these template files into your project folder.
3. **Edit each file as described below:**

### File Descriptions

- `research_paper.qmd`: Main Quarto manuscript. Follow the APA outline and section comments. Add your analysis and visualizations in R code chunks.
- `references.bib`: Bibliography file. Export your sources from Zotero and update this file.
- `README.md`: Project overview, abstract, and checklist. Follow the step-by-step instructions.
- `data/survey_data.csv`: Replace with your real survey data after collection. See `README_data.md` for details.
- `data/README_data.md`: Explains the data file format and privacy requirements.
- `output/`: Folder for your rendered PDF. Use Quarto to render your manuscript.

### Project Workflow

- Complete each section of the manuscript as you progress through the phases outlined in the checklist.
- Analyze your data in RStudio and create at least two APA-formatted visualizations.
- Render your manuscript to PDF and save it in the `output/` folder.
- Push all files to your GitHub repository and submit your PDF to Blackboard.
