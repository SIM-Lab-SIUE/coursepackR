# One-Time Environment Setup

Installs Quarto CLI support, TinyTeX for PDF rendering, and core R
packages needed for the course. Safe to run multiple times; only
installs missing components.

## Usage

``` r
mccourse_setup()
```

## Value

Invisibly, `TRUE`.

## Examples

``` r
if (FALSE) { # \dontrun{
mccourse_setup()
} # }
```
