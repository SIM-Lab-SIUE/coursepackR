# Data Folder Guide

This folder contains your survey data for the final project.

## survey_data.csv
- This is a dummy file to help you get started.
- Columns:
  - `participant_id`: Unique ID for each participant
  - `age`: Age of participant
  - `gender`: Gender (e.g., Male, Female, Other)
  - `social_media_use`: Numeric score (e.g., hours per day or scale)
  - `news_trust`: Numeric score (e.g., 1-5 Likert scale)
  - `political_knowledge`: Numeric score (e.g., 0-10 scale)

## Instructions
- Replace `survey_data.csv` with your real survey data after you close your survey.
- Make sure your columns match the expected format above.
- Do NOT include any personally identifiable information (PII) in your data file.
- Save your final data as `survey_data.csv` in this folder before rendering your manuscript.
