# Zotero Guide for APA Formatting

This guide explains how to use Zotero and Better BibTeX to manage your bibliography and export a `.bib` file for Quarto.

## Steps
1. Install Zotero and the browser connector
2. Install Better BibTeX plugin
3. Create a new collection for your project
4. Add sources and check metadata
5. Export your collection as a `.bib` file (Better BibTeX format, keep updated)
6. Link `references.bib` in your Quarto YAML header

For more help, see the technical guide or ask your instructor.
