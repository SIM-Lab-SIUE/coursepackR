# White Paper Project Template

This folder contains all the files you need to start your group white paper project.

## Project Structure
- `white_paper.qmd`: Main Quarto manuscript
- `references.bib`: Bibliography file
- `data/`: Survey data folder
    - `survey_data.csv`: Dummy data file
- `output/`: Folder for rendered PDF

## Step-by-Step Checklist
1. Fill in your project details in `white_paper.qmd`.
2. Replace dummy data in `data/survey_data.csv` with your real survey data.
3. Export your group bibliography from Zotero and update `references.bib`.
4. Write your manuscript following the outline and comments in `white_paper.qmd`.
5. Analyze your data in RStudio and create at least two visualizations.
6. Render your manuscript to PDF in the `output/` folder.
7. Push all files to your GitHub repository and submit your link.
