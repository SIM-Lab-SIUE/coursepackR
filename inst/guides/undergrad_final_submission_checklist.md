# Undergraduate Group White Paper: Final Submission Checklist

Use this checklist before submitting your MC451 group white paper. Each item is required for full credit.

## Manuscript & Analysis
- [ ] All sections completed in `white_paper.qmd` (journalistic outline)
- [ ] Two Research Questions (RQs) and/or Hypotheses (RHs) clearly stated
- [ ] At least two data visualizations created in R
- [ ] All data analyzed in RStudio; code and results included
- [ ] References formatted in APA or journalistic style
- [ ] Bibliography exported from Zotero to `references.bib`
- [ ] Survey instrument included in Appendix

## Data & Privacy
- [ ] Real survey data in `data/survey_data.csv` (no personal identifiers)
- [ ] `README_data.md` updated with data description and privacy notes

## Rendering & Output
- [ ] Manuscript rendered to PDF using Quarto
- [ ] Final PDF saved as `output/white_paper.pdf`

## GitHub Repository
- [ ] Private repository created (e.g., `white-paper-f25`)
- [ ] All project files pushed: manuscript, data, output, README, references
- [ ] Repository includes a clear `README.md` and checklist
- [ ] No sensitive data or personal info in the repository
- [ ] GitHub link submitted as part of final deliverables

## Submission
- [ ] Final PDF and GitHub link submitted as instructed
