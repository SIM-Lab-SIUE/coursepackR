# Zotero Guide for Group White Paper

This guide explains how to use Zotero and Better BibTeX to manage your group bibliography and export a `.bib` file for Quarto.

## Steps
1. All team members join the Zotero group library
2. Add sources and check metadata
3. Export your group collection as a `.bib` file (Better BibTeX format, keep updated)
4. Link `references.bib` in your Quarto YAML header

For more help, see the technical guide or ask your instructor.
